# AGENTS.md — מערכת המשימות המשותפת (Claude + Antigravity)

מסמך זה מסביר את שיטת העבודה האוטומטית של הפרויקט. **גם Claude וגם Antigravity צריכים לקרוא אותו ולפעול לפיו.**
המטרה: המשתמש (דור) כותב משימות במקום אחד, וכל סוכן מבצע רק את מה שתויג עבורו — בלי כפילויות ובלי לשבור את הפרויקט.

## הקבצים במערכת

- **TASKS.md** — קובץ הקלט. המשתמש כותב כאן משימות באזור "תיבת קלט". לא עולה קרדיט להוסיף.
- **AGENT_REPORT.docx** — דוח Word **דו-עמודי**: עמוד 1 = רשומות Claude, עמוד 2 = רשומות Antigravity. הרשומה החדשה תמיד בראש העמוד של הסוכן.
- **append_report.py** — סקריפט עזר שמוסיף רשומה לעמוד הנכון. שימוש: `python3 append_report.py <summary.txt> <claude|ag>`. Antigravity מריץ אותו עם הארגומנט `ag` כדי לכתוב לעמוד 2.
- **CLAUDE.md** — כללי הקוד והבנייה של הפרויקט. חובה לכבד אותם.

## תיוג משימות — מי מטפל במה

בתוך אזור "תיבת קלט" ב-TASKS.md, כל שורה לא-ריקה שאינה מתחילה ב-`#` היא משימה. התגית בתחילת השורה קובעת מי מבצע:

| תחילת שורה | מי מבצע |
|---|---|
| `claude:` | הסוכן האוטומטי של Claude |
| `ag:` או `antigravity:` | Antigravity |
| (ללא תגית) | ברירת מחדל — Claude |

**כל סוכן מבצע אך ורק את המשימות שתויגו עבורו, ומתעלם לחלוטין מהאחרות** (לא נוגע בהן, לא משנה אותן, משאיר אותן בתיבת הקלט).

## כללי עבודה לכל סוכן (כולל Antigravity)

1. **קרא קודם** את CLAUDE.md ואת TASKS.md.
2. **סנן** רק את המשימות של התגית שלך.
3. **חוק מכסה 80%:** לפני כל משימה חדשה, אם ניצלת ~80%+ מהמכסה — עצור, שמור מה שעשית, והשאר את השאר לפעם הבאה.
4. **אל תשבור כלום:** אחרי כל תיקון הרץ `npm run build` ו-`npm test`. אם נכשל בגלל השינוי שלך — בטל את השינוי והשאר את המשימה פתוחה.
5. **אל תעשה git commit / push.** רק השאר את הקבצים ערוכים.
6. **בסיום משימה מוצלחת:** מחק אותה מתיבת הקלט והעבר אותה לחלק "בוצע" ב-TASKS.md עם תאריך, התגית, וסיכום שורה.
7. **דווח:** הוסף רשומה ל-AGENT_REPORT.docx (Claude עושה זאת דרך append_report.py; Antigravity יכול לכתוב באותו אופן או לתעד בדרכו).
8. **כשלא בטוח — דלג ותעד.** עדיף לא לתקן מאשר לשבור.

## תזמון — הזמנים מפוצלים כדי ששני הסוכנים לא ירוצו יחד

- **Claude** רץ בשעות הזוגיות, כל שעתיים: 00:00, 02:00, 04:00, 06:00, 08:00, 10:00, 12:00, 14:00, 16:00, 18:00, 20:00, 22:00. מבצע רק משימות `claude:`. רץ רק כשאפליקציית Claude פתוחה; ריצה שהוחמצה נתפסת בפתיחה הבאה.
- **Antigravity** צריך לרוץ בשעות האי-זוגיות, כל שעתיים: 01:00, 03:00, 05:00, 07:00, 09:00, 11:00, 13:00, 15:00, 17:00, 19:00, 21:00, 23:00. מבצע רק משימות `ag:`.

בכל ריצה, כל סוכן: בודק מכסה (חוק 80%), אם אין מכסה מדלג ומנסה בריצה הבאה, ואם יש — מבצע את המשימות של התגית שלו לפי כללי העבודה למעלה.
